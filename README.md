📊 Waste Shipment Analysis Dashboard (Power BI)
🚀 Project Overview

This project focuses on analyzing waste generation and shipment data using Power BI to uncover meaningful insights and support data-driven environmental decision-making.

The dashboard provides a clear view of waste trends, top generators, and management methods through interactive visualizations.

🎯 Objectives
Analyze total waste generation across different sources
Identify top waste-generating companies
Understand waste distribution across categories
Examine waste management methods (landfill, incineration, etc.)
Enable interactive filtering for deeper insights
📈 Key Insights
📊 217M+ total waste analyzed across 913 generators
🏭 A small number of generators contribute significantly to total waste
📈 Waste trends show fluctuations with peak industrial periods
♻️ Different waste types follow different management methods
⚖️ Uneven distribution of waste highlights monitoring gaps
🛠️ Tools & Technologies
Power BI (Dashboard Development)
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
📊 Dashboard Features
KPI cards for quick overview
Year-wise waste trend analysis
Waste distribution by type
Waste management method analysis
Top 5 waste generators
Interactive filters for dynamic exploration
